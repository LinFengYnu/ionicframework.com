<h1 id="overview">组件</h1>

Ionic 应用程序是由被称为组件的高级构建模块所组成的。组件化的开发方式能让你得以快速地构建出应用程序的图形界面。Ionic 内置了大量的组件，包括模态框、弹出窗口，以及卡片等。请查阅下面的示例，看一看每个组件都长啥样，并学习如何使用这些组件。熟悉完基础知识之后，可以前往 [API 文档](../api)了解如何自定义组件。
