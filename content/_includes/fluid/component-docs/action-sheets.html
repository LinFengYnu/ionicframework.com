Action Sheets

改进这篇翻译 Action Sheets 从屏幕底部向上滑出，并显示一系列选项以便用户确认或者取消这次操作。Action Sheets 有时可以作为菜单的替代品，但是它们**不应该**用于导航功能。 Action Sheet 永远都显示在页面的其他组件之上，并且只有在关闭 Action Sheet 之后才能与页面中的其他内容进行交互。一旦触发 Action Sheet，页面中的其他内容都会变暗，以便将用户的视角聚焦到选项之上。 _若要获取更多信息，请查阅 [API 文档](../api/components/action-sheet/ActionSheetController)。_

基础用法

示例 示例源代码

```typescript import { ActionSheetController } from 'ionic-angular'; export class MyPage { constructor(public actionSheetCtrl: ActionSheetController) { } presentActionSheet() { let actionSheet = this.actionSheetCtrl.create({ title: 'Modify your album', buttons: [ { text: 'Destructive', role: 'destructive', handler: () => { console.log('Destructive clicked'); } },{ text: 'Archive', handler: () => { console.log('Archive clicked'); } },{ text: 'Cancel', role: 'cancel', handler: () => { console.log('Cancel clicked'); } } ] }); actionSheet.present(); } } ```
