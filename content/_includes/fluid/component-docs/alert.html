Alerts

改进这篇文档 Alerts 弹框不失为一种向用户提供单个特定操作，或者一系列操作选项的好方法。它们还可以为用户提示重要信息，或者要求用户做出一个（或多个）决定。 从 UI 的角度来看，Alerts 可以被当做一种只挡住一部分屏幕的“浮动”模态框。这意味着 Alerts 只能用于诸如密码验证、简单的应用提示等其他快速操作。而更加深入的用户操作流则应该保留给全屏的[模态框](#modals)。 Alerts 弹框是非常灵活，且易于定制的。 _若要获取更多信息，请查阅 [API 文档](../api/components/alert/AlertController)。_

基础用法

示例 示例源代码

Basic Alerts 通常用于向用户提示一些新增的信息（如应用中的改动以及新功能等），需要用户确认的紧急情况，或是告知用户操作成功与否的提示。 ```typescript import { AlertController } from 'ionic-angular'; export class MyPage { constructor(public alertCtrl: AlertController) { } showAlert() { let alert = this.alertCtrl.create({ title: 'New Friend!', subTitle: 'Your friend, Obi wan Kenobi, just accepted your friend request!', buttons: ['OK'] }); alert.present(); } } ```

Prompt Alerts

示例 示例源代码

Prompts 为用户提供了一种在 Alert 弹框里输入数据或信息的途径。在许多应用程序的“进一步操作”用户流中，经常会用到 Prompts 以便让用户输入密码。 ``` import { AlertController } from 'ionic-angular'; export class MyPage { constructor(public alertCtrl: AlertController) { } showPrompt() { let prompt = this.alertCtrl.create({ title: 'Login', message: "Enter a name for this new album you're so keen on adding", inputs: [ { name: 'title', placeholder: 'Title' }, ], buttons: [ { text: 'Cancel', handler: data => { console.log('Cancel clicked'); } }, { text: 'Save', handler: data => { console.log('Saved clicked'); } } ] }); prompt.present(); } } ```

Confirmation Alerts

示例 示例源代码

Confirmation Alerts 通常被用在当用户想要在应用程序中执行某个特定操作时，提示用户是否确定要执行该操作。一个典型的使用场景：当用户想要在通讯录里面删除一个联系人时，Confirmation Alert 会提示用户是否确定要这么做。 ``` import { AlertController } from 'ionic-angular'; export class MyPage { constructor(public alertCtrl: AlertController) { } showConfirm() { let confirm = this.alertCtrl.create({ title: 'Use this lightsaber?', message: 'Do you agree to use this lightsaber to do good across the intergalactic galaxy?', buttons: [ { text: 'Disagree', handler: () => { console.log('Disagree clicked'); } }, { text: 'Agree', handler: () => { console.log('Agree clicked'); } } ] }); confirm.present(); } } ```

Radio

示例 示例源代码

Radio Alerts 可以被看做是一种特殊的 Confirmation Alert，其弹框中的选项是以 [Radio](https://ionicframework.com/docs/components/#radio) 组件的形式呈现的。Radio Alerts 向用户提供了一系列选项，但是在这些选项中你只能选中一个。 ``` import { AlertController } from 'ionic-angular'; export class MyPage { constructor(public alertCtrl: AlertController) { } showRadio() { let alert = this.alertCtrl.create(); alert.setTitle('Lightsaber color'); alert.addInput({ type: 'radio', label: 'Blue', value: 'blue', checked: true }); alert.addButton('Cancel'); alert.addButton({ text: 'OK', handler: data => { this.testRadioOpen = false; this.testRadioResult = data; } }); alert.present(); } } ```

Checkbox

示例 示例源代码

Checkbox Alerts 同样是一种特殊的 Confirmation Alert，其弹框中的选项是以 [Checkbox](/docs/components/#checkbox) 组件的形式呈现的。Checkbox Alerts 向用户提供了一系列选项，但是在这些选项中你可以选择一个或多个。 ``` import { AlertController } from 'ionic-angular'; export class MyPage { constructor(public alertCtrl: AlertController) { } showCheckbox() { let alert = this.alertCtrl.create(); alert.setTitle('Which planets have you visited?'); alert.addInput({ type: 'checkbox', label: 'Alderaan', value: 'value1', checked: true }); alert.addInput({ type: 'checkbox', label: 'Bespin', value: 'value2' }); alert.addButton('Cancel'); alert.addButton({ text: 'Okay', handler: data => { console.log('Checkbox data:', data); this.testCheckboxOpen = false; this.testCheckboxResult = data; } }); alert.present(); } } ```
