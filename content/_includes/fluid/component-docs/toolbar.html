Toolbar

改进这篇文档 toolbar 是一个通用栏，可以在应用程序中用作页眉，子页眉，页脚甚至子页脚。由于 `ion-toolbar` 基于 flexbox，所以无论你的页面中有多少个工具栏，它们都将正确显示，`ion-content` 也会相应调整。 注意：与导航一起使用时，通常在 `ion-header` 中使用 [NavBar](../api/components/navbar/Navbar)。 _有关更多信息，请查看 [API 文档](../api/components/toolbar/Toolbar)。_

改变颜色

你可以通过更改元素的属性来更改 toolbars 的颜色。 ```typescript @Component({ template: ` Toolbar Toolbar Toolbar Toolbar ` }) ``` 你也可以用相同的方式更改导航栏的颜色。这将允许你在应用中为每个页面的导航栏使用不同的颜色。 ```html Dark Danger Secondary ```

Toolbars 中的按钮

改进文档 按钮可以添加到页眉和页脚 toolbars。要将一个按钮添加到 toolbars，我们需要先添加一个 `ion-buttons` 组件。该组件包装一个或多个按钮，并且可以给出 `start` 或 `end` 属性用来控制它包含按钮的位置： ```html Send To...

Ash, Misty, Brock

 ```

Toolbars 中的 Segment

改进文档 [Segments](#segment) 是允许用户在不同数据集之间切换的好方式。使用以下标记将 segment 添加到 toolbar 中： ```html New Hot ```

Toolbars 中的 Searchbar

另一个常见的设计范例是在 toolbar 中包含一个搜索栏。 ```html {% raw %}{{ item }}{% endraw %} ```
