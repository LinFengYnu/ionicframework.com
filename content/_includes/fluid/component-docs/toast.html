Toast

改进文档 Toast 是一种微妙的的通知，它出现在应用内容的顶部。通常情况下，Toasts 将在短时间内显示，然后自动解除。 _有关更多信息，请查看 [API 文档](../api/components/toolbar/ToastController)。_

基本用法

Demo Demo 源码

```typescript import { ToastController } from 'ionic-angular'; export class MyPage { constructor(public toastCtrl: ToastController) { } presentToast() { let toast = this.toastCtrl.create({ message: 'User was added successfully', duration: 3000 }); toast.present(); } } ```
