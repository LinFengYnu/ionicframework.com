Checkbox

改进这篇文档 多选框（checkbox）是一种用于保存 *布尔值* 的输入（input）组件。Ionic 的 Checkbox 组件与原生 HTML 的 checkbox 并无区别。然而就像其他的 Ionic 组件一样，Checkbox 组件在不同的平台中的样式会有所区分。你可以使用 `checked` 属性来设定默认值，而 `disabled` 属性则被用来禁止用户选择或取消选择某个特定的值。 _若要获取更多信息，请查阅 [API 文档](../api/components/checkbox/Checkbox)._
