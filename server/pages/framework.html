用你熟悉并喜爱的 Web 技术构建应用程序

知道如何构建网站？那么你就已经知道如何构建移动应用了。 Ionic 框架为其提供了最好的 web 和原生应用组件， 用于构建高度互动的原生和渐进 Web 应用。

  • 原生和 web 优化

    Ionic 模拟原生应用 UI 指南并使用原生 SDK， 将原生应用的 UI 标准和设备特性与开放 web 的全部功能和灵活性结合在一起。 Ionic 使用 Cordova 或 Phonegap 进行本地部署，换而言之可以作为渐进式 Web 应用在浏览器中运行。

一个完整的移动工具包，为 Web 开发人员构建。

你所需知道的开始创建功能全面的移动应用程序的一切，只需几分钟。

  • 一个简单的 CLI 用于在任何平台上创建，构建，测试和部署你的 Ionic 应用程序。 这就像魔法，但它的确不是。
  • Ionic 原生通过在 TypeScript 中包装 Cordova 插件来解锁原生 API 和功能。
  • 实时重载是因为在开发的每个编译和重新部署应用步骤中都是一大块工作量。
  • Ionicons图标包包含数百种最常见的应用程序图标。 MIT 授权，开箱即用。
  • Deeplinking 你可以从 web 链接启动你的应用，从一开始就加载特定的视图。
  • AoT 编译意味着你的应用能够快速加载。 不要在加载屏幕上闪烁，那样你会错过它。
